# O-Count Statistics StashApp Plugin

This repository contains the **O-Count Statistics** plugin for the organizer [Stash](https://stashapp.cc/). It Provides locally computed, statistical insights and visualizations instance under `<stash-url>/stats`.

This plugin requires the **StashGraphs** and **CommunityScriptsUILibrary** plugins.

## Configuration

The visibility of individual charts can be toggled via the plugin settings.

-   **Enable O-Count by Tags Chart:** If enabled, the chart showing O-Count distribution by tags will be displayed.
-   **Enable O-Count by Year Chart:** If enabled, the chart showing O-Count distribution by year of media will be displayed.
-   **Enable O-Count by Performer Chart:** If enabled, the chart showing O-Count distribution by performer will be displayed.
-   **Enable O-Count by Studio Chart:** If enabled, the chart showing O-Count distribution by studio will be displayed.
-   **Enable O-Count vs. Play Count Scatter Chart:** If enabled, the scatter chart comparing O-Count and Play Count will be displayed.
-   **Enable O-Count vs. Play Duration Scatter Chart:** If enabled, the scatter chart comparing O-Count and Play Duration will be displayed.
-   **Enable Top O-Count Items Chart:** If enabled, the chart showing the top O-Count items will be displayed.
-   **Enable Scene O-Count Distribution Chart:** If enabled, the pie chart showing scene O-Count distribution will be displayed.
-   **Enable Image O-Count Distribution Chart:** If enabled, the pie chart showing image O-Count distribution will be displayed.
