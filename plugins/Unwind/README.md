# Unwind StashApp Plugin

This repository contains the **Unwind** plugin for the organizer [Stash](https://stashapp.cc/). It provides a "year in review" summary of your Stash activity.

This plugin requires the **StashGraphs** and **CommunityScriptsUILibrary** plugins.

## Features

-   **Year in Review:** Provides a comprehensive summary of your Stash activity.
-   **Year Selection:** A dropdown menu allows you to select any year for which play-count or O-count data is available. The report will dynamically update when you select a new year.
